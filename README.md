# nodejs-chatbot
a chatbot using openai, socket io and express in nodejs

How to start:

clone this repo

in .env file, paste your openai api key in OPENAI_API_KEY variable

open the terminal
install the dependencies using:
npm install

run the code by:
nodemon app.js

open the browser, go to
http://localhost:2000/
